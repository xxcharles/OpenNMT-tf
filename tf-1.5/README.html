Documentation¶

We currently use Sphinx to automatically build documentation from the code.

Install dependencies¶

pip install sphinx
pip install sphinx_rtd_theme
pip install recommonmark

Register sources for autodoc¶

# To run only if a Python source file is added, removed, or renamed.
rm -f docs/package/*
sphinx-apidoc -e -M -o docs/package opennmt opennmt/tests

Build locally¶

cd docs/ && make html
